Primero veremos la forma general de declarar una clase, con atributos, constructor y sus respectivos metodos.
```ts
TypeScript
class Hero{
	name:string
	life:number
	atributte:string
	damage:number
	constructor(name:string,life:number,atributte:string,damage:number){
	  this.name=name
	  this.life=life
	  this.atributte=atributte
	  this.damage=damage
	}
	
}
```
